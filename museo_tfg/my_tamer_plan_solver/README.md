## Instalación de Tamer

Ejecutar la siguentes líneas en la terminal (dentro del entorno creado si es necesario)
 
1. Instalar UP

```bash
pip install unified-planning
```

2. Instalar planificador tamer

``` bash
pip install unified-planning[tamer]
```
