TFG

## Estructura del proyecto

Este repositorio contiene el código del TFG, incluyendo:

- `tfg-ia/` - Experimentos y código relacionado con IA y planificación
- `museo_tfg/` - Paquetes ROS2 para el sistema del museo
- `my_small_house/` - Entorno de simulación
- `plantilla-latex-tfg-tfm/` - Plantilla LaTeX para la memoria

### Nota histórica

El código de `tfg-ia/` fue originalmente un repositorio independiente que se integró mediante Git Subtree para simplificar el desarrollo. Ahora se mantiene directamente desde este repositorio.
