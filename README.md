## Estructura del proyecto

Este repositorio contiene el código del TFG, incluyendo:

- `llm_planners/` - Experimentos y código relacionado con IA y planificación
- `museum_navigation/` - Paquetes ROS2 para el sistema del museo
- `my_small_house/` - Entorno de simulación
- `plantilla-latex-tfg-tfm/` - Plantilla LaTeX para la memoria

### Nota histórica

El código de `llm_planners/` fue originalmente un repositorio independiente que se integró mediante Git Subtree para simplificar el desarrollo. Ahora se mantiene directamente desde este repositorio.
